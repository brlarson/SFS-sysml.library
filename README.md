# SFS-sysml.library
Supplemental Formal Semantics for KerML and SysML v2 Libraries (sysml.library).  The particular libraries for SFS are in sysml.library/KernelLibraries/Kernel Semantic Library/SFS library.  The standard libraries can be found at Systems-Modeling/SysML-v2-Release.

The SFS version of sysml.library was developed using Syside from Sensmetry.

It can be used with Syside, but you must redirect from the standard sysml.library by including the following line in your syside.toml file, edited for the path to sysml.library on your computer:
std = "/Users/brianlarson/git-sysml.library/sysml.library"
